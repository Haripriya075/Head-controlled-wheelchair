## Overview

This project presents a **head-controlled wheelchair prototype** developed as an undergraduate mini project. The system enables individuals with mobility impairments to control wheelchair movement through natural head movements, providing a hands-free assistive mobility solution.

A 3-axis accelerometer-based tilt sensor mounted on a wearable cap detects the user's head movements in real time. The sensor data is processed by an Arduino microcontroller, which controls DC motors through an L298N motor driver to move the wheelchair in the corresponding direction.

## Problem Statement

Many individuals with severe physical disabilities are unable to operate conventional wheelchairs using manual controls. This project aims to provide an affordable and hands-free mobility solution using head movement as the control mechanism.

---

## Objectives

- Develop a low-cost assistive wheelchair prototype.
- Detect head movements using a 3-axis accelerometer.
- Process sensor data using Arduino.
- Control wheelchair movement through motor drivers.
- Provide an intuitive hands-free navigation system.

## Direction Mapping

| Head Movement | Wheelchair Action |
|--------------|-------------------|
| Tilt Forward | Move Forward |
| Tilt Backward | Move Backward |
| Tilt Left | Turn Left |
| Tilt Right | Turn Right |
| Neutral Position | Stop |
